## What is Sass?
[sass-lang.com](http://sass-lang.com/documentation/file.SASS_REFERENCE.html)
- Sass is an extension of CSS.
- Allows the use of 
  - variables
  - nested rules
  - mixins
  - inline imports,
  - and more
